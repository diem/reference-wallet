# LRW Frontend

This project is based on [React](https://github.com/facebook/react)

## How to use?

1. Download or clone this repo.

2. Install dependencies.

```bash
npm install
// or
yarn install
```

3. Run project on Browser.

```bash
npm run start
// or
yarn start
```

## Run tests

```bash
npm run test
// or
yarn test
```
